Typo3-restclient ui

A REST client backend module for TYPO3 CMS

Download as .zip Download as .tar.gz View on GitHub

Introduction

This extension provides a backend module for sending HTTP requests and receiving HTTP responses from a RESTful API resource.
You can set many options by using Typo3-Backend, or even you can choose to use the restclient extension settings.
The CRUD operations GET, POST, PUT, DELETE are supported.

Requirements

It requires TYPO3 6.2, PHP 5.3, and restclient extension (>=1.1.2).
